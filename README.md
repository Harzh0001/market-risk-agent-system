# Market Risk Agent System

Multi-agent Rai-aligned market risk prediction system deployable on Railway.

## Run local tests
```bash
python -m tests.test_orchestrator
python -m tests.test_backtester
```

## Deploy
Push to GitHub and connect Railway to the repo. Set `PORT` automatically from Railway.
